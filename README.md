# CPP_practizes

Я задался целью стать С++ разработчиком, имея опыт разработки на С (стажировка, учебный опыт) и python (учебный опыт). 
Этот язык программирования мне понравился своим мощным фукнционалом. С ним я смогу гибко работать с памятью компьтера, при этом имея 2 парадигмы: ООП(что очень важно для Software) и функционального. Мне нравится его развитие: Новые спецификации говорят о его движении вперед и исправлению некоторых трудностей. И самое главное - область применения. Software, а также есть используется в системном программировании, Backend, Сетевом программировании, оптимизации.
