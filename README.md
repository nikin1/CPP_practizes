## Проекты:
 - 3D Viewer. (Софт на QT для отображения 3D моделей при чтении файла)
Проект представляет собой программу для просмотра 3D каркасных моделей из файлов формата .obj. Реализованы функции загрузки модели, отображения в виде проволочного каркаса, аффинных преобразований (перемещение, вращение, масштабирование). Архитектура построена по паттерну MVC с применением объектно-ориентированного подхода на C++. В интерфейсе предусмотрена настройка проекций, стиля отображения ребер и вершин, изменение цвета фона, создание скриншотов и gif-анимаций текущих трансформаций модели.




 - SmartCalc (Софт на QT с графическим интерфейсом калькулятора, а также вывод графика фукнций через Plot lib)
Приложение для вычисления математических выражений, включая обратную польскую нотацию с реализацией по алгоритму Дейкстры, а также финансовых расчетов (кредитный и депозитный калькуляторы). Backend реализован на C и ориентирован на эффективное выполнение вычислений. Проект разделен на Backend и Frontend для оптимизации и четкого разграничения функций. Компиляция и сборка автоматизированы через Makefile

## Сейчас выполняю:
- Выполнение курсовой работы по Параллельному программированию (OpenMP).
Разработка высокопроизводительного C++ приложения для анализа ChIP-seq данных с целью поиска регуляторных мотивов ДНК. Реализованы алгоритмы сопоставления мотивов с учетом расширенного IUPAC-кодирования, обеспечен подсчет числа совпадений и частотный анализ. Использованы параллельные вычисления на базе MPI+OpenMP для обработки больших объемов биоинформационных данных
